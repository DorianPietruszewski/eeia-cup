# EEIA CUP

Nowoczesny szkielet strony konkursowej w Next.js przygotowany pod wydarzenie EEIA CUP.

## Start

1. Zainstaluj zależności:

```bash
npm install
```

2. Uruchom aplikację lokalnie:

```bash
npm run dev
```

3. Otwórz projekt w przeglądarce na `http://localhost:3000`.

## Co już jest

- Next.js z App Routerem
- TypeScript
- Tailwind CSS
- Sekcje startowe dla hero, informacji, harmonogramu, FAQ i kontaktu
- Miejsce na późniejsze dodanie grafik i treści eventowych

## Następny krok

Jeśli chcesz, mogę teraz przygotować wersję bardziej „eventową” wizualnie albo dodać strukturę pod konkretne mechaniki gry konkursowej.
